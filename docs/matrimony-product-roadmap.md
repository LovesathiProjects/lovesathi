# Lovesathi Matrimony Product Roadmap

This keeps the remaining matrimony-specific product gaps visible after the first luxury UI pass.

## Priority 1: Premium Product Depth

- Curated matches dashboard with daily recommendations, compatibility reasons, and profile-completion nudges.
- Biodata-style profile dossier with printable/shareable family-safe view.
- Interest-request flow with statuses such as sent, accepted, declined, and withdrawn.
- Family involvement options for parent/sibling-managed profiles.
- Saved searches and shortlist folders for family review.
- Rich compatibility signals for age, height, education, career, location, culture, lifestyle, family values, and partner preferences.

## Priority 2: Trust, Safety, And Verification

- Admin verification queue for ID, face scan, and profile review.
- Real KYC provider integration or a manual approval workflow with audit logs.
- Report moderation queue with actions, notes, and status changes.
- Profile quality checks for missing photos, unsafe text, duplicate accounts, and incomplete biodata.
- Private photo reveal controls and contact visibility controls.

## Priority 3: Monetization

- Real payment gateway integration for India-friendly checkout.
- Subscriptions table, entitlement checks, plan expiry, invoices, refunds, and admin subscription view.
- Premium feature gates for advanced filters, who-shortlisted-you, profile priority, and concierge support.

## Priority 4: Notifications And Communication

- In-app notification center.
- Email/SMS notifications for interests, accepted requests, matches, verification status, and profile reminders.
- Supabase Realtime monitoring and production chat delivery checks.
- Safer message moderation and block/report visibility.

## Priority 5: Launch Polish

- Replace remaining plain utility screens with the luxury design system.
- Add FAQ content, richer contact/support flows, and legal review-ready copy.
- Confirm local env points to Lovesathi Supabase, not Affinity, before local QA.
- Add analytics for onboarding completion, discovery actions, shortlist, interests, and payment intent.
